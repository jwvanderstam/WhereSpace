## Technical Details

### CSS Classes
- `.nav-bar` - Container with gradient background
- `.nav-link` - Individual navigation items
- `.nav-link.active` - Current page highlight
- `.nav-icon` - Emoji icons
- Smooth animations on hover

### Behavior
- Active page highlighted automatically
- All links functional (go to respective pages)
- Scrollable on narrow screens
- Custom scrollbar styling
